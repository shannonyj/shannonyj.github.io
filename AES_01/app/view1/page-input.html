Plaintext and Key Input

AES-128 allows a fixed block size and a fixed key size as 128 bits.
It is termed as state which comprises of 16 bytes (4 x 4 matrix of bytes),
where each byte is represented using two HEX characters.
Hence, inputs from 00 to ff are allowed.

Input

This is required.
Input exactly 2 hex characters.
Must be within 00 to ff.

Key

This is required.
Input exactly 2 hex characters.
Must be within 00 to ff.
